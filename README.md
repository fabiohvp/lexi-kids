# 🎒 Lexi Kids

O **Lexi Kids** é um web app educacional e interativo projetado para auxiliar crianças em fase de alfabetização a aprenderem a grafia de palavras comuns (animais e objetos) em Português (PT-BR). Com um design lúdico, colorido e repleto de micro-animações, o jogo transforma o aprendizado da escrita em uma experiência divertida e motivadora.

---

## ⭐ Recursos Principais

- 🔊 **Feedback por Áudio Nativo**: Integração com a API de Síntese de Voz do navegador (`speechSynthesis`) com velocidade (rate) e tom (pitch) ajustados para serem amigáveis e acolhedores para crianças.
- 📈 **Progressão Dinâmica de Níveis**: O jogo possui mais de 8 níveis de dificuldade que se adaptam conforme a pontuação da criança, aumentando o comprimento das palavras e a quantidade de letras ocultas a serem preenchidas.
- 🦁 **Grande Banco de Dados**: Mais de 300 palavras cadastradas entre animais e objetos, representadas por emojis de alta definição que dispensam dependências de APIs de imagens externas.
- 🧠 **Navegação Inteligente**:
  - Foco automático no próximo campo vazio ao digitar a letra correta.
  - Atalho de teclado (Barra de Espaço) para ouvir a pronúncia a qualquer momento.
  - Suporte inteligente para a tecla `Backspace` para corrigir letras anteriores.
- 🎨 **Design Kids-Friendly**: Cores vibrantes, sombras suaves estilo "cartoon", e feedbacks visuais imediatos (verde para acertos, vermelho com pulsação para erros).

---

## 🛠️ Tecnologias Utilizadas

- **Gemini** (Este jogo foi completamente Vibe Coded)
- **React** (Vite)
- **Tailwind CSS** (Estilização responsiva e lúdica)
- **Lucide React** (Ícones modernos)
- **Web Speech API** (Síntese de voz nativa e offline)

---

## 🎮 Como Jogar

1. Clique em **JOGAR!** na tela inicial para iniciar.
2. Observe o ícone (emoji) da palavra e sua categoria (Animal ou Objeto).
3. Caso queira ouvir a pronúncia da palavra, clique no botão **Ouvir Palavra** ou aperte a **Barra de Espaço**.
4. Digite as letras que faltam nos blocos amarelos.
5. Acerte todas as letras para somar pontos, subir de nível e avançar para o próximo desafio!
